Fingerspelling_Detection.xlsx is the annotations for the fingerspelling detection task.

Video_Clip_Name	: sign video clip name (folder: Auslan_Daily/Auslan_Daily xx/Signer Only Video Clip/Signer/).
Begin_Frame	: beginning frame of current fingerspelling.
End_Frame	: ending frame of current fingerspelling.
Finger_Spelling	: current fingerspelling.
split		: data split [train, dev, test].
